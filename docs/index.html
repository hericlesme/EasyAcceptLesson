  • Get Started

  • O que é o EasyAccept?

    O EasyAccept é uma ferramenta desenvolvida para ajudar a desenvolver equipes para criar testes de acessibilidade. Tais testes são de caixa preta e visam capturar os requisitos funcionais de um sistema de software, conforme expresso por um cliente.

  • Onde Posso baixar o EasyAccept?

    Você pode baixar a última versão do EasyAccept através deste link pelo sourceforge. Ou pode baixar o .JAR diretamente clicando aqui.

  • Dúvidas Frequentes

  • Como eu devo adicionar o EasyAccept ao meu projeto?

    É bastante simples. Clique com o botão direito no seu projeto, procure por "Build Path" e selecione a opção "Configure Build Path...". Em seguida, selecione "Libraries" e aperte em "Add JARs..." ou "Add Externals JARs...".
    É recomendado que você não adicione JARs externos ao seu projeto, pois isto pode causar uma eventual quebra de dependência, se você acessar em outro local. Para que isto não aconteça, coloque o arquivo JAR do EasyAccept dentro do seu projeto, e o referencie internamente.

  • Já adicionei a ferramenta ao meu projeto... Mas como a executo?

    Para executar o EasyAccept você deve fazer o seguinte.

    
	public static void main(String[] args) {
		args = new String[] { "SampleFacade", "acceptance_tests/us1.txt" };
		EasyAccept.main(args);
	}

    - "Sample Facade" corresponde à sua Facade. (Se estiver usando pacotes, referencie também com pacotes. Exemplo: pacote.Facade)

    - "acceptance_tests/us1.txt" corresponde ao script de teste, que está localizado na pasta acceptance_tests. (Lembre-se também de referenciar o caminho correto para seus scripts de teste)

  • "Ei, meu teste dá dando um erro aqui, acho que tá errado..."
